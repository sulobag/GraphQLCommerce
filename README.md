## E-Ticaret Backend API (GraphQL + Laravel + Lighthouse)

Bu proje, kurumsal ölçekte bir e-ticaret platformu için yalnızca GraphQL tabanlı bir arka uç API geliştirmek amacıyla hazırlanmış bir örnek uygulamadır. Ön yüz kapsam dışı tutulmuş olup tüm odak; güçlü bir domain yapısı, ölçeklenebilir servisler, kararlı veri modeli ve modern PHP/Laravel ekosisteminde temiz bir mimari oluşturmaktır.

Proje; Laravel + Lighthouse, PostgreSQL, Redis, Elasticsearch, Passport, Nginx ve Docker Compose bileşenleri ile uçtan uca çalışır durumda tasarlandı.
